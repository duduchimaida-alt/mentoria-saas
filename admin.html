Integração Google Agenda

Cola aqui a URL que você gerou ao publicar o Apps Script como Web App. Sem ela, ao marcar uma call só vai abrir o link manual do Google Agenda.

Setup Google Agenda em 5 minutos

  1. Acessa script.google.com/home e faz login com a conta Google onde tu quer os eventos.
  2. Clica em Novo projeto. Vai abrir um editor de código.
  3. Apaga o código padrão e cola o script abaixo: 
    function doPost(e){
  try{
    var data = JSON.parse(e.postData.contents);
    var cal = CalendarApp.getDefaultCalendar();
    var start = new Date(data.start);
    var end = new Date(data.end || (new Date(data.start).getTime() + 60*60*1000));
    var event = cal.createEvent(data.title || 'Call', start, end, {
      description: data.description || '',
      location: data.location || ''
    });
    return ContentService.createTextOutput(JSON.stringify({
      success: true,
      eventId: event.getId(),
      url: 'https://calendar.google.com/calendar/event?eid=' + Utilities.base64Encode(event.getId() + ' ' + cal.getId()).replace(/=+$/, '')
    })).setMimeType(ContentService.MimeType.JSON);
  } catch(err){
    return ContentService.createTextOutput(JSON.stringify({success:false, error:err.toString()})).setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(){
  return ContentService.createTextOutput(JSON.stringify({ok:true, message:'GOAT webhook ativo'})).setMimeType(ContentService.MimeType.JSON);
}
  4. Renomeia o projeto pra GOAT Mentoria - Agenda (canto superior esquerdo).
  5. Clica em Implantar → Nova implantação (botão azul, canto superior direito).
  6. Engrenagem ⚙️ ao lado de "Selecionar tipo" → escolhe Aplicativo da Web.
  7. Preenche:
    • Descrição: v1
    • Executar como: Eu (teu email)
    • Quem tem acesso: Qualquer pessoa
  8. Clica em Implantar → Google vai pedir permissões. Aceita tudo (acesso ao teu Google Agenda).
  9. Vai aparecer uma URL do aplicativo da Web tipo https://script.google.com/macros/s/AKfyc.../exec. Copia ela.
  10. Volta no painel e cola a URL no campo da janela de configuração. Clica em TESTAR CONEXÃO pra validar.
💡 Depois de configurado: toda call que você salvar no painel cria automaticamente o evento no teu Google Agenda, sem nenhum clique extra.
